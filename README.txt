Description des documents présents dans le projet

Binômes:
    Gabriel DUVAL
    Yves YAMADJAKO

tables.sql
    Code qui génère les tables de la base de donnée - le fichier peut être lancé automatiquement 
    depuis le terminal PostgresSQL. Certaines tables sont aussi commentées pour décrire les contraintes
    mise en place dans la base.
ajout_data.sql
    Code qui remplit automatiquement les tables à partir de fichiers csv présents dans le fichier 
    Tableaux_CSV. Le dossier Tableaux_CSV doit impérativement être dans le même repertoire que ce 
    fichier.
requetes.sql 
    Code pour les requêtes SQL à implémenter. Celles-ci peuvent être copiées-collées une par une dans
    un terminal PostgresSQL pour tester leurs effets. Chaque requêtes est etiquetée par un commentaire
    pour spécifier son effet.
Tableaux_CSV
    Dossier avec les fichiers csv à insérer dans la base - il devrait y en avoir 21. Chaque fichier porte
    le nom de la table qu'il remplit. Il faut noter que la plupart de ces données ont été générées avec 
    l'aide de ChatGPT pour des fins illustratives, donc ne sont pas forcément toujours représentatives 
    d'une veritable base de données.
ProjetBdD_DUVAL_YAMADJAKO
    Rapport en format PDF qui décrit la modélisation de la base de données, ainsi que ses limites et des
    améliorations possibles.
